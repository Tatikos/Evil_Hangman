Evil Hangman Game

Introduction

This project implements an "evil" version of the Hangman game where the computer tries to maximize its chances of winning by dynamically selecting the secret word based on the player's guesses.

Algorithm

The algorithm works by:

  1. Reading a dictionary of words of the same length
  2. Starting with all possible words
  3. After each guess, dividing words into families based on letter positions
  4. Selecting the largest family to maximize difficulty
  5. Continuing until the player wins or runs out of guesses

Modules

  • hashtable.c : Data structures (lists and hashtables)
  • dictionary.c : Dictionary reading and management
  • game.c : Game logic and flow control
  • as2.c : Main program entry point
